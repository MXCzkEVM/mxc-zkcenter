# Testing with hardhat



## Tested development environment

- Ubuntu 22.04

- nodejs v20.18.1

- npm 10.8.2

- yarn 1.22.22



## Setup

After clone the repos, run this once to install the npm packages.

```
yarn
```



Compile the contract.

```
yarn compile
```



Clean all output files.

```
yarn clean
```



Test on local with harthat.
```
yarn test
```
